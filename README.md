# Rondelson Car - Reposicionamento Visual

Página estática para reposicionamento visual da Rondelson Car, com foco em estoque premium, avaliação de usados, financiamento e contato por WhatsApp.

## Estrutura

- `index.html`: conteúdo principal e seções da página.
- `styles.css`: identidade visual, responsividade e componentes.
- `script.js`: filtros do estoque, simulador visual e formulários para WhatsApp.
- `package.json`: scripts opcionais para rodar com Vite.

## Rodar localmente

Abra `index.html` no navegador ou use:

```bash
npm install
npm run dev
```

## Publicação

Como o site é estático, pode ser publicado em GitHub Pages, Vercel ou Netlify. As imagens do estoque estão referenciadas a partir do site atual da Rondelson Car/Autoconf.
