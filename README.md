# Rondelson Car Rebrand

Landing page estatica para reposicionamento visual da Rondelson Car, com foco em estoque premium, avaliacao de usados, financiamento e contato por WhatsApp.

## Estrutura

- `index.html`: conteudo principal e secoes da pagina.
- `styles.css`: identidade visual, responsividade e componentes.
- `script.js`: filtros do estoque, simulador visual e formularios para WhatsApp.
- `package.json`: scripts opcionais para rodar com Vite.

## Rodar localmente

Abra `index.html` no navegador ou use:

```bash
npm install
npm run dev
```

## Deploy

Como o site e estatico, pode ser publicado em GitHub Pages, Vercel ou Netlify. As imagens do estoque estao referenciadas a partir do site atual da Rondelson Car/Autoconf.
